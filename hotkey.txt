This is the file where you specify what key you want to press to save that dope song you're listening to

Our default hotkey is above and should be on the first line.

If you change this, then you need to restart the program for changes to take place

To check that your custom hotkey works, hover over the checkmark icon in your task tray
And if it failed it will say "OMFG ERROR"

You can have as many "modifiers" as you want but you can only have 1 key
Example: 
This is okay: alt shift ctrl f7
This is not okay: alt f7 f8
